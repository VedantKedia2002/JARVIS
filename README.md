🎙️ CRISTY - Personal AI Voice Assistant

CRISTY is a Python-based personal voice assistant developed to automate everyday computer tasks using voice commands. It combines speech recognition, text-to-speech, and external APIs to provide a hands-free desktop assistant experience.

---

✨ Features

- 🎤 Voice command recognition
- 🔊 Text-to-Speech responses
- 🌐 Open websites using voice commands
- 🧮 Open Calculator
- 📷 Open Camera
- 😂 Tell jokes
- 📰 Fetch latest news
- ⏰ Tell current date and time
- 🔍 Search information online
- ⚡ Execute common desktop automation tasks

🛠️ Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- PyWhatKit
- PyAudio
- Requests
- News API
- Wikipedia API
- Webbrowser
- OS Module
